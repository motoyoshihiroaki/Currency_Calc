## スクリプトの説明
円、ドル、ポンド、ユーロの相関計算をするスクリプトです。\n
半角で数字を入力してエンターキーを押すと計算します。

## API情報の管理
API情報は、`.env`ファイルで管理します。
`.env.sample`ファイルを参考に記述してくだしさい。
